Detailed Description

Library to solve stochastic delay differential equations. The library uses polymorphism to design an SDDE model (class modelSDDE) from building blocks. Those building blocks are the delayed vector field (class vectorFieldDelay), a stochastic vector field (class vectorFieldStochastic). and an SDDE numerical scheme (class stochasticNumericalSDDE).
