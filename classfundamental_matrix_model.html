fundamentalMatrixModel Class Reference

Numerical fundamental matrix model. A fundamentalMatrixModel is defined by a linearized vector field and a numerical scheme. The current state of the fundamentalMatrixModel is also recorded as a fundamental matrix. Attention: the constructors do not copy the vector field and the numerical scheme given to them, so that any modification or freeing will affect the fundamentalMatrixModel. More...
